### 跨平台文件管理器

![image-20230610203819162](https://bib0.com/xc/i/2023/06/10/image-20230610203819162.png)

🆔  资源名称：[📁Spacedrive - 一个来自未来的文件浏览器](https://www.spacedrive.com/)

⭐️  资源描述：

>📄Spacedrive 是一个基于 Rust 的开源跨平台文件管理器，支持Windows/Linux/iOS/ Android
>
>它采用了虚拟分布式文件系统，通过一个 API 来操作访问多设备上的文件
>
>它可以管理分布在多端的文件，无论是云端还是离线硬盘，进而生成你独有的个人分布式云

🌐 Github资源地址：[点击跳转](https://github.com/spacedriveapp/spacedrive)

### 相册同步网络

![image-20230610082157795](https://bib0.com/xc/i/2023/06/10/image-20230610082157795.png)

🆔  资源名称：fregie/pho

⭐️  资源描述：替代手机上的自带相册应用，将手机上的照片同步到网络存储。

>计划支持的网络存储有：
>
> - Samba
> - Webdav
> - NFS
> - 夸克网盘
> - 百度网盘
> - oneDrive
> - google drive
> - google photo
>
>**PS：目前只有安卓版本**

🌐 Github资源地址：[点击跳转](https://github.com/fregie/pho)

### 大麦网自动购票程序

![image-20230610080758828](https://bib0.com/xc/i/2023/06/10/image-20230610080758828.png)

🆔  资源名称：🎟dm-ticket - 大麦网自动购票程序

⭐️  资源描述：

> 📄项目支持 Docker 部署以及 Rust 环境部署
>
> 购票前需要在配置文件中填写演唱会信息、选座方式、购票数量等相关参数
>
> 完成配置后，只需要执行简单的命令，DM Ticket 就可以自动模拟用户行为进行购票操作

🌐 Github资源地址：[点击跳转](https://github.com/ClassmateLin/dm-ticket)
