视频漫画媒体容器 APP

🆔 资源名称：🎬媒体盒子 - 一个开源的视频漫画媒体容器 APP

⭐️ 资源描述：

📄这是一个把媒体网站插件化了的观影 APP，直接把网站的内容调用到 APP ，除此再无广告等其他干扰观看的内容

下载这个 APP 之后需要去「插件仓库」下载插件才能阅观看，目前提供了7个插件，其中一个是看新闻的，其他则是影视插件

🗣上手体验了一下，手感比较丝滑，界面也非常简约，缺点是插件暂时还不够丰富

🌐 资源地址：点击跳转 | 🧩Github地址

模组化素材管理

🆔 资源名称：🧰mCollection - 一款开源的模组化素材管理软件

⭐️ 资源描述：

📄它最大的亮点就是支持数据库管理、支持扩展插件、部署私人素材库，一定程度上可以平替 eagle

它还同时支持 MacOS 和 Windows ，由于是开源的，并且刚问世不久，所有有些BUG，不过相信会越来越好

🧩插件列表看了下，有Eagle&Billfish导入、开发助手、媒体标注、SD文生图、3D文件预览、代码支持、相识图片查找、阿里云盘同步等实用插件

无代码可视化爬虫

🆔 资源名称：🕷️EasySpider - 一个开源的无代码可视化爬虫工具

⭐️ 资源描述：

📄软件由浙江大学学生开发，版权归浙大所有，并且还有论文支持

下载打开软件之后，你只需要在网页上选择自己想要爬的内容并根据提示框操作即可完成爬虫设计和执行即可

📝使用体验

小白看一遍教程基本能会个8成以上，所以受众面非常广泛

智能程度感觉不如后羿、八爪鱼这些商业软件，但是作为开源软件已经完全合格了

跨平台文件管理器

🆔 资源名称：📁Spacedrive - 一个来自未来的文件浏览器

📄Spacedrive 是一个基于 Rust 的开源跨平台文件管理器，支持Windows/Linux/iOS/ Android
