匿名视频群聊

⭐️ 描述：🔮安全的直接视频群聊

隐私是这个项目背后的中心思想。仅使用开放技术（例如WebRTC ），该技术适用于所有现代浏览器。因此，无需安装额外的软件，并且该方法仍然面向未来。

可以通过Brie.fi/ng立即免费开始对话。还提供免费的 iOS 应用程序。

提供桌面共享、文本聊天和简单邀请链接等功能。借助 Javascript 和 Vue 的知识，可以轻松添加更多内容。
