使用用国内服务的可直接申请TrustAsia CA证书 如：宝塔、腾讯云、阿里云等，证书到期后重新申请即可。

Cloudflare

📁 网站简介：全球知名网站，只要将域名解析cf上，并开启cloudflare的cdn就可以直接使用免费证书，无需做其他设置。

🔗 资源地址：点击跳转

ZeroSSL

📁 网站简介：为每个账号提供3个免费90天的SSL安全证书，可免费续期，支持ACME Certificates自动签发、自动续期。ZeroSSL最大特色是可以免费为纯IP地址申请签发SSL安全证书，其它支持纯IP地址签发的SSL安全证书基本都是要付费的。ZeroSSL为全英文网站，如实用上有任何问题，可以参考阿拉丁神灯ZeroSSL教程 ❤

🔗 资源地址：点击跳转

SSLforfree

📁 网站简介：免费申请Let’s Encrypt证书 可申请通配符 SSL 证书（即申请一个证书可用于所有该域名下所有子域名证书）
