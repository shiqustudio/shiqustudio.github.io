Git 常用指令汇总

 xucong 发布于 2018-10-30 收录于 Git
  • 工作区：就是你在电脑里能看到的目录。
  • 暂存区：英文叫 stage, 或 index。一般存放在 “.git 目录下” 下的 index 文件（.git/index）中，所以我们把暂存区有时也叫作索引（index）。
